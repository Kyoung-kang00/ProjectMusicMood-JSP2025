# MusicMood-Project2025 - 음악 스트리밍 서비스 플랫폼

#### Team MusicMood (음악 스트리밍 서비스 플랫폼)
# 💻 프로젝트 소개
자바를 처음 접하여 어려운 진입장벽을 낮춰주기 위한 입문자용 코딩 학습 플랫폼

# 🕒 개발기간
* 25.06.03 ~ 25.07.11

# 👥 멤버구성
김정민 - API명세서, DB명세서 작성
         문제제출 및 정답여부, 시험풀이 및 채점
         MonacoEditer적용 및 정답검증 시스템 구축

박준채 - 커뮤니티 페이징 및 검색 정렬
         글 등록, 수정, 삭제 댓글 등록, 삭제
         관리자 페이지 데이터 통계 호출
         
강경원 - SpringSecurity적용 및 소셜 로그인 구현
         사용자 관리(로그인, 회원가입) - > SpringSecurity로그인 인증
         문제목록 페이징 검색, 정렬, 카테고리 일괄 적용

# 개발환경
* `Java17`
* **DataBase**: MySQL
